// Button Type: SWITCH (Toggle)
// =============================
// A SWITCH button maintains its state when clicked.
// It toggles between ON and OFF with each click.
//
// Instructions:
// 1. In the Default Scene, I0.0 should be set to SWITCH type (default)
// 2. Click I0.0 - it turns ON and stays ON
// 3. Click again - it turns OFF and stays OFF
// 4. The output Q0.0 follows the switch state
//
// This is useful for:
// - Main power switches
// - Mode selection toggles
// - Enable/disable controls

// Simple SWITCH example: Switch controls output
LD I0.0       // Load switch state
OUT Q0.0      // Output follows switch

// Switch with indicator
LD I0.1       // Load second switch
OUT Q0.1      // Light follows switch

// Switch-controlled timer
LD I0.2       // Load switch
TON T0, 30    // Start timer when switch ON (3 seconds)
LD T0         // Load timer done bit
OUT Q0.2      // Output activates after 3 seconds
